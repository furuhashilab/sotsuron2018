## 基本形式
* 卒論本文は GitHub の公開リポジトリとして作成する。
* Markdown, Wiki, GitHub Pages 等、本文の表現手法は同一のリポジトリ内であれば問わない。
* IMRAD形式で整理すれば、文字数や表現方法は問わない
